# Torres Intelligence — Landing Page

Landing page institucional da Torres Intelligence, desenvolvida em React + Vite.

## 🚀 Tecnologias

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui
- React Router DOM

## ⚙️ Rodando localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:5173

## 🏗️ Build de produção

```bash
npm run build
npm run preview
```

## ☁️ Deploy na Vercel

O projeto já está configurado com `vercel.json` para funcionar corretamente como SPA (Single Page Application).

1. Faça push para o GitHub
2. Conecte o repositório em [vercel.com](https://vercel.com)
3. Clique em **Deploy** — sem configurações adicionais necessárias

## 📁 Estrutura

```
src/
├── components/
│   ├── landing/     # Seções da LP
│   └── ui/          # Componentes base (shadcn)
├── pages/           # Páginas da aplicação
└── lib/             # Utilitários
```
